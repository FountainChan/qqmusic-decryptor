# 元数据批量处理优化方案

## 📋 方案信息

- **方案日期**：2026-01-31
- **方案编号**：V1.0
- **目标**：优化元数据处理流程，提高效率和资源利用率

---

## 1. 问题分析

### 1.1 当前工作流程

**CLI 版本（main_cli.py）：**
```
逐个文件解密
    ↓
解密完成一个文件
    ↓
立即调用 add_flac_metadata()（Line 465）
    ├─ 提取音轨号并写入
    ├─ 调用 API 获取专辑信息
    ├─ 下载封面
    ├─ 嵌入封面到文件
    └─ 写入发行年份
    ↓
继续下一个文件
```

**GUI 版本（gui_backup/main_gui.py）：**
- 类似的逐个处理逻辑

**补充脚本（supplement_album_metadata.py）：**
- 支持批量处理整个目录
- 已实现完整的元数据功能（音轨号、封面、年份）
- 具有专辑级别的 API 缓存

### 1.2 存在的问题

| 问题 | 影响 | 严重程度 |
|------|------|----------|
| API 调用频繁 | 每个 FLAC 文件都会触发 API 调用 | 🔴 高 |
| 资源浪费 | 同一专辑的歌曲重复获取相同的专辑信息和封面 | 🟠 中 |
| 效率低下 | 逐个处理导致大量 IO 操作 | 🟠 中 |
| 代码重复 | `main_cli.py` 和 `supplement_album_metadata.py` 功能重叠 | 🟡 低 |
| 实际使用中需二次处理 | 用户经常需要再次运行 `run_supplement.sh` | 🟠 中 |

### 1.3 资源浪费分析

**场景**：解密一个专辑的 10 首歌曲

**当前流程（逐个处理）**：
- API 调用次数：10 次（每个文件一次）
- 封面下载次数：10 次
- 总耗时：~50 秒（假设每次 5 秒）

**优化流程（批量处理）**：
- API 调用次数：1 次（整个专辑一次）
- 封面下载次数：1 次
- 总耗时：~5 秒（单次处理）
- **节省时间**：~45 秒（90% 效率提升）

## 2. 优化方案

### 2.1 方案选择

**选择方案 A：完全替换**

**理由**：
- ✅ 批量处理明显更高效
- ✅ 减少代码维护成本
- ✅ 用户操作流程更合理（先解密，后统一补充元数据）
- ✅ `supplement_album_metadata.py` 已具备完整功能（包括音轨号提取）

### 2.2 新工作流程

**CLI 版本（main_cli.py）：**
```
逐个文件解密（不进行元数据修改）
    ↓
所有文件解密完成
    ↓
统一调用 supplement_album_metadata.py 处理整个输出目录
    ├─ 按专辑分组
    ├─ 每个专辑只调用一次 API
    ├─ 批量处理所有歌曲的音轨号、封面、年份
    └─ 保存 cover.jpg
```

**GUI 版本（gui_backup/main_gui.py）：**
- 同样移除逐个文件的元数据修改
- 在解密完成后调用 `supplement_album_metadata.py`

### 2.3 方案架构

```
解密流程（main_cli.py 或 GUI）
    ↓
批量元数据流程（supplement_album_metadata.py）
    ├─ 步骤1：按专辑分组
    ├─ 步骤2：获取专辑信息（每专辑一次 API 调用）
    ├─ 步骤3：批量处理所有文件
    │   ├─ 音轨号（每文件一次）
    │   ├─ 年份（每文件一次）
    │   └─ 封面（每文件一次）
    └─ 步骤4：保存 cover.jpg（每专辑一次）
```

```python
def run_supplement_metadata(self):
    """
    运行元数据补充脚本
    
    处理整个输出目录，批量添加音轨号、封面、年份
    """
    import subprocess
    
    try:
        # 检查 supplement_album_metadata.py 是否存在
        script_path = os.path.join(os.path.dirname(__file__), 'supplement_album_metadata.py')
        
        if not os.path.exists(script_path):
            self.log(f"元数据脚本不存在: {script_path}", "WARNING")
            return
        
        # 构建命令
        cmd = ['python', script_path, self.output_dir]
        
        self.log(f"执行命令: {' '.join(cmd)}", "INFO")
        
        # 运行脚本
        result = subprocess.run(
            cmd,
            capture_output=True,
            text=True,
            encoding='utf-8'
        )
        
        if result.returncode == 0:
            self.log("元数据处理成功", "INFO")
        else:
            self.log(f"元数据处理失败: {result.stderr}", "ERROR")
        
    except Exception as e:
        self.log(f"执行元数据脚本异常: {e}", "ERROR")
```

### Phase 4: 更新脚本和文档

#### 步骤 4.1：修改启动脚本

**文件**：`run_supplement.sh`

**修改内容**：
```bash
#!/bin/bash
# ============================================================
# 专辑元数据补充工具 - Shell 脚本版本
# ============================================================

# 设置脚本目录（本文件的父目录）
SCRIPT_DIR="$(cd "$(dirname "$0")" && pwd)"

# 自动检测输出目录（优先级从高到低）
# 1. 命令行参数
# 2. config.ini 中的 output_dir
# 3. 默认目录
INPUT_DIR="$1"

if [ -z "$INPUT_DIR" ]; then
    # 从 config.ini 读取输出目录
    CONFIG_FILE="$SCRIPT_DIR/config.ini"
    if [ -f "$CONFIG_FILE" ]; then
        OUTPUT_DIR=$(grep -A 5 "\[PATHS\]" "$CONFIG_FILE" | grep "output_dir" | cut -d'=' -f2 | xargs)
        if [ -n "$OUTPUT_DIR" ] && [ -d "$OUTPUT_DIR" ]; then
            INPUT_DIR="$OUTPUT_DIR"
            echo "使用配置文件中的输出目录: $INPUT_DIR"
        fi
    fi
    
    # 仍然为空，使用默认目录
    if [ -z "$INPUT_DIR" ]; then
        INPUT_DIR="$SCRIPT_DIR/../Decrypted"
        INPUT_DIR="$(cd "$INPUT_DIR" && pwd)"
        echo "使用默认输出目录: $INPUT_DIR"
    fi
else
    echo "使用指定的目录: $INPUT_DIR"
fi

# 检查目录是否存在
if [ ! -d "$INPUT_DIR" ]; then
    echo "============================================================"
    echo "   错误：目录不存在"
    echo "============================================================"
    echo ""
    echo "目录路径: $INPUT_DIR"
    echo ""
    exit 1
fi

# 运行 Python 脚本
echo "============================================================"
echo "   专辑元数据补充工具"
echo "============================================================"
echo ""
echo "处理目录: $INPUT_DIR"
echo ""

python "$SCRIPT_DIR/supplement_album_metadata.py" "$INPUT_DIR" "$@"

# 检查执行结果
if [ $? -neq 0 ]; then
    echo ""
    echo "============================================================"
    echo "   处理失败"
    echo "============================================================"
    exit 1
fi
```
