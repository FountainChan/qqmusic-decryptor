## 🔍 根本原因分析

### 问题根源

**代码错误**（修复前）：
```python
# 修复前（错误）
if cover_data:
    save_cover_to_directory(album_dir, cover_data)
```

**参数分析**：
- `album_dir`：歌手目录（`G:/QQMusic/Decrypted/VipSongsDownload/侧田`）
- `audio_files[0]`：第一个音频文件（`侧田\From Justin...\01 B_O_K.ogg`）

**问题**：
- `save_cover_to_directory(album_dir, cover_data)` 直接保存到 `album_dir`
- 没有考虑音频文件所在的目录（专辑子目录）
- 导致封面保存到了歌手目录，而不是专辑子目录

### 修复方案

**代码修改**（修复后）：
```python
# 修复后（正确）
if cover_data:
    # 获取第一个音频文件所在的目录（专辑子目录）
    first_audio_dir = os.path.dirname(audio_files[0])
    
    # 保存封面
    cover_path = save_cover_to_directory(first_audio_dir, cover_data)
    
    # 添加详细日志
    logger.info(f"封面已保存到: {cover_path}")
```

**关键改进**：
1. ✅ 使用 `os.path.dirname(audio_files[0])` 获取专辑子目录
2. ✅ 保存封面到专辑子目录，而不是歌手目录
3. ✅ 添加详细日志，显示封面保存的实际路径
