## ⚙️ 自定义配置

### 修改默认目录

1. **右键编辑**
   - 右键 `run_supplement.bat`
   - 选择 "编辑"

2. **修改目录路径**
   - 找到第 17 行
   - 修改 `set DEFAULT_INPUT_DIR=G:\QQMusic\Decrypted`
   - 改为你的路径

3. **保存文件**
   - 按 `Ctrl+S` 保存

### 支持的路径格式

- ✅ **绝对路径**：
  - Windows 路径：`G:\QQMusic\Decrypted`
  - UNC 路径：`\\Server\Share\Music`

- ✅ **相对路径**：
  - 相对于脚本目录：`.\Decrypted`
  - 相对于父目录：`..\Decrypted`

- ✅ **带空格的路径**：
  - 自动处理，无需引号
  - 示例：`G:\My Music\Decrypted Songs`

## 🔧 故障排除

### 问题 1：双击后立即退出

**可能原因**：
- 默认目录不存在
- 默认目录是文件而不是目录

**解决方法**：
1. 检查默认目录路径是否正确
2. 确认目录存在
3. 确认是目录（不是文件）

### 问题 2：找不到 Python

**错误信息**：
```
错误：未找到 Python
请确保 Python 已安装并添加到 PATH 环境变量中

下载地址：https://www.python.org/downloads/
```

**解决方法**：
1. 下载并安装 Python 3.6+
2. 将 Python 添加到 PATH
3. 重启命令提示符
4. 重新运行脚本

### 问题 3：处理失败

**错误信息**：
```
处理失败，错误代码：1
```

**可能原因**：
- API 连接失败
- FLAC 文件缺少标签
- 目录权限不足

**解决方法**：
1. 检查网络连接
2. 检查 API 服务是否运行
3. 检查 FLAC 文件是否包含 ARTIST 和 ALBUM 标签
4. 检查目录读写权限
