## 二、简化后的统计结构

### 文件级统计
```python
file_stats = {
    "total": 0,           # 总音频文件数（FLAC + OGG）
    "processed": 0,       # 成功处理年份/封面的文件数
    "unchanged": 0,       # 无需修改的文件数（已有或无数据）
    "failed": 0           # 处理失败的文件数
}
```

### 操作级统计（简化版）
```python
operation_stats = {
    "year_cover": {           # 年份和封面合并统计
        "success": 0,         # 成功写入年份和/或封面
        "skipped": 0,         # 无API数据跳过
        "failed": 0           # 写入失败
    },
    "cover_saved": {          # 封面文件保存统计
        "saved": 0,          # 成功保存的专辑目录数
        "failed": 0          # 保存失败的目录数
    }
}
```

### 统计规则定义

#### 文件级统计规则
| 状态 | 定义 | 示例 |
|------|------|------|
| **processed** | 文件成功写入年份和/或封面 | FLAC文件写入年份和封面 |
| **unchanged** | 无需修改的文件（已有完整元数据 或 无数据可补充） | 所有元数据已存在 或 无albummid |
| **failed** | 文件处理过程中发生异常 | 文件损坏无法读取 |

#### 操作级统计规则
| 操作类型 | 计数条件 | 说明 |
|---------|---------|------|
| **year_cover.success** | 成功写入年份和/或封面 | FLAC嵌入封面，OGG写入年份 |
| **year_cover.skipped** | 无API数据无法写入 | 无albummid或网络问题 |
| **year_cover.failed** | 写入失败 | 文件写入异常 |
| **cover_saved.saved** | 成功保存cover.jpg到专辑目录 | 每个有封面的专辑目录+1 |
| **cover_saved.failed** | 保存失败 | 文件系统错误 |

            operation_stats["year_cover"]["success"] += 1
